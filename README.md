# Insulin-Gene-Mutation-Analysis
A bioinformatics pipeline using Biopython to fetch, analyze, and visualize the Human Insulin Gene (INS). Includes genomic stats, protein translation, and motif search.

# Human Insulin Gene Analysis

## Overview
This project performs a fundamental bioinformatics analysis on the Human Insulin Gene (INS) using Python. It fetches real-time data from NCBI, calculates genomic statistics, translates DNA to Protein, and visualizes amino acid distribution.

##  Features
* NCBI Integration: Automatically fetches gene data using `Biopython`.
* Genomic Stats: Calculates Nucleotide length and GC content.
* Translation: Converts DNA coding sequences into Protein sequences.
* Motif Search: Identifies specific DNA patterns (K-mers).
* Data Visualization: Generates charts for amino acid frequencies.

##  Technologies
* Python 
* Biopython
* Matplotlib
